# cursor-config

Личный репозиторий с конфигурацией для Cursor (и рядом стоящих инструментов), которую удобно:

- видеть в workspace как обычные файлы,
- хранить в git,
- подключать обратно через симлинки в стандартные директории.

В этом репо **версионируются только “человеческие” артефакты**:

- `skills/` — пользовательские skills (`~/.cursor/skills` → симлинк сюда)
- `agents/` — пользовательские subagents (`~/.cursor/agents` → симлинк сюда)
- `rules/` — общие правила/шпаргалки, которые не хочется держать в конкретном проекте (при необходимости можно тоже симлинкать)

То, что **не стоит** тащить в git:

- `~/.cursor/projects/`, `plugins/`, `plans/`, `ide_state.json` (локальный state)
- `~/.cursor/mcp.json` (может содержать чувствительные настройки/токены)
- всё, что похоже на ключи/секреты (`*.pem`, `*.key`, `.env` и т.п.)

## Быстрая установка (Linux)

```bash
ln -s /src/cursor-config/skills  ~/.cursor/skills
ln -s /src/cursor-config/agents  ~/.cursor/agents
```
